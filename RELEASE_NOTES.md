# Initial Clean Release v1.0.0

## 中文

这是 TIPS Prompt Manager 的首次干净发布。项目已从桌面快捷方式指向的 Python/Tkinter 脚本整理为可维护源码、测试、构建脚本和可发布资产。

### 下载说明

- `tips-prompt-manager-v1.0.0-windows-x64.exe`：Windows x64 单文件离线提示词管理器。
- `tips-prompt-manager-v1.0.0-windows-x64.zip`：便携 ZIP，包含 EXE、README、LICENSE 和第三方声明。
- `SHA256SUMS.txt`：发布资产的 SHA256 校验值。

程序支持中英文界面、本地密码验证、分类管理、搜索高亮和一键复制。EXE 未进行数字签名；作者、版权、产品名和版本元数据已写入。请校验 SHA256 后使用。

## English

This is the first clean release of TIPS Prompt Manager. The app has been organized from the Python/Tkinter script launched by the desktop shortcut into maintainable source code, tests, build scripts, and release assets.

### Downloads

- `tips-prompt-manager-v1.0.0-windows-x64.exe`: Windows x64 single-file offline prompt manager.
- `tips-prompt-manager-v1.0.0-windows-x64.zip`: Portable ZIP with the EXE, README, LICENSE, and third-party notices.
- `SHA256SUMS.txt`: SHA256 checksums for release assets.

The app supports Chinese/English UI, local password verification, category management, highlighted search, and one-click copy. The EXE is not digitally signed; author, copyright, product, and version metadata are present. Verify SHA256 before use.
