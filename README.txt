TIPS Prompt Manager / TIPS 提示词管理器

中文说明请见 README.md。
English documentation is available in README.md.

快速启动：
1. 双击桌面的 TIPS 快捷方式，或运行“启动提示词管理器.bat”。
2. 首次启动会要求设置本地密码。
3. 提示词、分类、语言偏好和认证状态都保存在本机。

Quick start:
1. Double-click the TIPS desktop shortcut or run "启动提示词管理器.bat".
2. Set a local password on first launch.
3. Prompts, categories, language preference, and authentication state are stored locally.
